How to run the queue

1. Build the queue here. Export queue.json.

2. Place your API key:

echo 'key_...' > godot/tools/.runway_key

3. Run:

python3 godot/tools/runway_render.py queue.json --dry-run
python3 godot/tools/runway_render.py queue.json --limit 1
python3 godot/tools/runway_render.py queue.json
